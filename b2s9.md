### Problema 1
**Sincronización de Sensores**

Descripción: Simula dos sensores que emiten valores periódicos:

- Sensor1: Rango [20°C, 30°C], cada 1 segundo.
- Sensor2: Rango [30°C, 40°C], cada 1.5 segundos.
- Usa zip para combinar los valores de ambos sensores en un solo flujo.
- Detén el flujo después de 10 pares de valores.
- Genera una alerta si la diferencia absoluta entre las lecturas de Sensor1 y Sensor2 es mayor a 5°C.
  

```mathematica
Sensor1: 22°C, Sensor2: 27°C, Difference: 5°C
Sensor1: 24°C, Sensor2: 34°C, Difference: 10°C
🚨 ALERT! Significant difference detected: 10°C
```
### Solucion problema 1
```scala
package Semana1.TallerS1

import rx.lang.scala.Observable
import scala.util.Random
import scala.concurrent.duration.*

object Sensores extends App {
  val sensor1 = Observable.interval(1.second).map(_ => Random.between(20, 30))
  val sensor2 = Observable.interval((1.5).second).map(_ => Random.between(20, 30))

  val combinacion = sensor1.zip(sensor2)
    .take(10)
    .subscribe(
      onNext = { case (s1,s2) =>
        val diferenciaAbs = Math.abs(s1 - s2)
        if(diferenciaAbs > 5) println(s"🚨 ALERT! Significant difference detected\n Diferencia detectada: $diferenciaAbs" )
        else println(s"Sensor1: $s1, Sensor2: $s2, Diferencia: $diferenciaAbs")
      }
    )
  Thread.sleep(20000)
}
```

### Problema 2
**Sistema de Alerta Complejo**

Descripción: Simula tres sensores que emiten datos periódicos:

- Sensor1: Temperatura (20°C a 30°C).
- Sensor2: Humedad (40% a 60%).
- Sensor3: Presión (950 hPa a 1050 hPa).
- Combina los datos de los tres sensores y detecta si alguna de las métricas está fuera de rango.
- Detén el flujo después de 20 combinaciones.

```yaml
Temperature: 22°C, Humidity: 50%, Pressure: 970 hPa
Temperature: 25°C, Humidity: 42%, Pressure: 1055 hPa
🚨 ALERT! Pressure out of range: 1055 hPa
...
```
### Solucion problema 2
```scala
package Semana1.TallerS1

import rx.lang.scala.Observable
import scala.util.Random
import scala.concurrent.duration.*

object SensorPeriodicos extends App {
  //Genera valores que pueden estar tanto dentro como fuera de rango
  val sensor1 = Observable.interval(1.second).map(_ => Random.between(10, 50))
  val sensor2 = Observable.interval(1.second).map(_ => Random.between(20, 80))
  val sensor3 = Observable.interval(1.second).map(_ => Random.between(850, 1060))

  val combinacion = sensor1.zip(sensor2).zip(sensor3).map {
      case ((s1, s2), s3) => (s1, s2, s3)
    }
    .zipWithIndex
    .take(20)
    .subscribe(
      onNext = { case ((s1,s2,s3),index) =>
        val num = index + 1
        println(s" $num ) Temperature: $s1, Humidity: $s2, Pressure: $s3")
        if(s1 <= 20 || s1 >= 30) println(s"🚨 ALERT! Temperature out of range: $s1")
        if(s2 <= 40 || s2 >= 60) println(s"🚨 ALERT! Himidity out of range:$s2")
        if(s3 <= 950 || s2 >= 1050) println(s"🚨 ALERT! Pressure out of range:$s3")
        println()

      }
    )
  Thread.sleep(40000)
}
```
